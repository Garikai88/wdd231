Site Plan: MobiBank+

Site Name

MobiBank+

The name "MobiBank+" was selected because it emphasizes a mobile-first type approach to modern banking. The prefix "Mobi", implies the aspect of portability and ease of use which is on the go. On the other hand the "+" symbol signifies the "extra" value-added services that the app provides. This includes an integrated financial tracking and enhanced security features.

Site Purpose

The purpose of MobiBank+ is to provide a secure and a streamlined digital platform for users in order for them to manage their personal finances. The site serves as a hub for users to view real-time account balances, perform instant peer-topeer transfers, and to access financial literacy resources to help them reach their savings goals.

Scenarios

  • How can I securely transfer money toa friend's account instantly using my phone?
  • Where can I find tools to help me set and track a monthly savings goal for a new car?

Color Scheme

The following colors will be used to establish brand trust and clarity:

  • Primary (Trust Blue): #003366- To be used for all main headings and navigation backgrounds.
  • Secondary (Growth Green): #2E8B57- Used for action buttons like "Open Account."
  • Background: #FFFFFF - Used for main body text to ensure maximum readability.

Typography

  • Heading Font: Roboto (Sans-serif) - Selected for its modern, authoritative feel.
  • Body Font: Open Sans (Sans-serif) - Selected for high legibility on small mobile screens.

Wireframes

The home page will focus on a clean, card-based interface that prioritizes account balances and quick actions.

Mobile View

A single-column layout with a top-heavy focus on current balances and a vertical list of transactions for easy scrolling on small screens.

Desktop View

A multi-column dashboard featuring a left-hand navigation sidebar and a detailed central grid for managing complex financial data and history.
